    About Mora Jai Puzzle Solver

    Welcome to the Mora Jai Puzzle Solver! This tool helps you solve Mora Jai puzzles by allowing you to set up the initial grid state and providing step-by-step solutions.

    How to Use:

    1. Set the corner colors using the colored squares in each corner
    2. Click on grid tiles to cycle through colors
    3. Click "Solve Puzzle" to get step-by-step solution
    4. Save your puzzles using the save feature
    5. Load previously saved puzzles from the dropdown

    Features:

    • Interactive grid setup
    • Step-by-step solution visualization
    • Save and load puzzle states
    • Random grid generation

    Keyboard Shortcuts:

    Hover over a tile or corner and press the following keys to change colors:

    • O - Orange
    • Y - Yellow
    • B - Blue
    • K - Black
    • G - Gray
    • P - Pink
    • U - Purple
    • W - White

    Tip: You can also use Tab to navigate between tiles and corners, then use the keyboard shortcuts to change colors.
